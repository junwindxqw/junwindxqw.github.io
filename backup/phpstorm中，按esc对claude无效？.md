使用node安装了Claude code，并配置调用GLM4.7

在phpstorm中打开终端，进入 Claude code 中后，使用了一些指定后，我想回到输入框中， 提示我 esc to exit 即可，
但是按esc，没有反应， 其实是phpstorm拦截了原因。

到配置中关闭： 

设置 -> 工具 -> 终端 -> 取消勾选 “使用Escape将焦点移至编辑器”
